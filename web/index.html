Ver pasos Comandos rápidos

Flujo completo

  1. Entrena o carga tu modelo PixelCNN++. Prepara un dataset consistente y genera muestras con coherencia visual.
  2. Genera imágenes. Filtra las mejores y usa keyframes claros para mantener estilo.
  3. Prepara el job de Flux.1. Define prompt, fps y fuerza de guía de imagen.
  4. Postprocesa. Interpola y aplica color grading si necesitas consistencia.

Comandos rápidos

1) Generar imágenes

python -m pixelcnn_flux1_pipeline generate-images \
  --config configs/pixelcnn_config.json

2) Preparar job de video

python -m pixelcnn_flux1_pipeline make-video \
  --config configs/flux1_config.json

3) Ejecutar todo

./scripts/run_pipeline.sh

Artefactos generados

  • Imágenes: outputs/pixelcnn_samples/
  • Metadata: outputs/pixelcnn_samples/metadata.json
  • Job de video: outputs/flux1_job.json
